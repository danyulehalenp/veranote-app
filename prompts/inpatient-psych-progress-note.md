Transform the source input into an inpatient psychiatry progress note.

Required section order:
1. Date / Interval Update
2. Symptom Review
3. Medications / Changes / Side Effects
4. Mental Status / Observations
5. Insight / Judgment
6. Safety / Risk
7. Assessment
8. Plan

Template goals:
- produce a clean inpatient psych progress note
- preserve explicit dates and current-day interval context
- keep symptom, safety, and plan language close to the provider's source note
- sound like a cleaned-up provider progress note, not a broad narrative summary
- support continued inpatient level of care when the source supports it
- preserve ongoing acute symptom burden without softening it into generic improvement language

Rules:
- Use only information documented in today's source input.
- Mental Status / Observations is required for inpatient psych notes.
- Build the MSE section from any supported interview, clinician, nursing, staff, collateral, or objective observations in the source, not just an explicitly labeled MSE block.
- If only limited MSE elements are supported, write a limited MSE and explicitly note that additional MSE details were not documented in the provided source.
- If prior-day content is included for reference, do not document it as if it occurred today unless today's note explicitly reaffirms it.
- Preserve current symptom persistence when directly stated.
- Do not infer improvement or deterioration unless documented.
- Do not invent side effects, tolerability, or medication response unless directly supported.
- Preserve direct inpatient risk/discharge language when it is documented.
- If the source supports that inpatient treatment remains necessary, preserve that logic without exaggerating beyond the source.
- Keep assessment brief and source-faithful.
- If patient denial conflicts with observed behavior or nursing/clinician observations, keep both sides visible and unresolved in the Assessment rather than choosing a cleaner interpretation.
- Keep behavioral observations attributed to the observer when that reduces false certainty.
- Do not convert observed internal preoccupation into confirmed hallucinations, and do not treat denial plus odd behavior as fully resolved absence either.
- Avoid adjudicating phrases such as "continues to exhibit," "consistent with active hallucinations," "supported by nursing observation," or equivalent wording when the observed behavior and the patient report remain in tension.
- Keep plan close to the actual documented plan.
- If the source includes direct statements about insight, engagement, groups, milieu, discharge readiness, or substance-use attitudes, preserve those points without softening them.
- Support single-paragraph provider style when the source suggests that note shape.
- Avoid arrows and non-ASCII formatting.

Writing guidance:
- Use compact paragraph-style prose within each section unless the source clearly points toward a one-paragraph progress-note style.
- Preserve direct clinical wording when possible.
- Prefer provider-note voice over polished model-summary voice.
- Avoid padded transitions and explanatory filler.
- If the source is already close to a usable progress note, clean it lightly rather than recasting it.
