Apply user-selected style controls while still obeying all no-fabrication and template-specific rules.

Controls:
- Output style: Concise, Standard, or Polished
- Format: Paragraph Style, Labeled Sections, or Minimal Headings
- Keep closer to source wording
- Flag missing info

Style behavior guidance:
- Concise: keep the note tight and avoid elaboration.
- Standard: balanced brevity and readability.
- Polished: allow slightly smoother wording, but do not add new facts or interpret beyond the source.
- Keep closer to source wording: preserve more of the original phrasing when clinically reasonable.

Important:
- For psychiatry notes, default behavior should already lean toward source-faithful wording and structure.
- If Keep closer to source wording is enabled, strongly prefer conservative rewriting over paraphrased interpretation.
- Preserve explicit dates and timeline markers from the source regardless of selected style.
- Within each section, prefer compact paragraph prose over fragment-style phrasing.
- Do not use the selected style as permission to invent, soften, strengthen, embellish, or generalize unsupported content.
- If Flag missing info is off, return an empty flags array unless a major ambiguity still needs surfacing for safe review.
